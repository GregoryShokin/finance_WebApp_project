# Фича — Инструкции «где взять выписку»
#бэклог #высокий-приоритет #импорт #онбординг
> Для каждого банка из whitelist — мини-инструкция со скриншотами «как скачать выписку в правильном формате».
---
## Проблема (2026-05-03)
Главный technical drop-off на онбординге — юзер не знает, где в его банке скачать выписку CSV/PDF/XLSX. У каждого банка интерфейс и название пунктов меню разные.

## Планируемое
### Контент (для каждого банка whitelist)
- Тинькофф (Т-Банк): веб + приложение
- Сбер: веб + СберБанк Онлайн
- Альфа-Банк
- ВТБ
- Райффайзен
- Озон Банк
- Яндекс Банк
- (Остальные — по мере добавления экстракторов)

Формат каждой инструкции:
- 3-5 шагов с скриншотами (скриншоты с маскированием PII!)
- Указание правильного формата (CSV vs PDF) и периода (месяц / 3 месяца)
- Tip «если получилось не то — попробуйте другой формат»
- Время прохождения «~2 минуты»

### Хранение
- Markdown-файлы в `frontend/content/bank-instructions/{bank_code}.md` или JSON-структура
- Скриншоты в `frontend/public/bank-instructions/{bank_code}/`
- Альтернатива: storage в БД (`bank_instruction` table) с админкой для редактирования

### UI
- На странице `/import` (пустой state) — кнопка «Где взять выписку?» → модалка с табом по выбранному банку
- В bank-picker рядом с каждым банком — иконка ℹ с открытием инструкции
- На странице банка-расширения (если выбран unsupported банк) — общие советы

## Критичность
**Высокий приоритет** — без этого онбординг ломается на технической детали.
