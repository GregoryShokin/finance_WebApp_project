# Фича — Вклад привязан к подушке
#бэклог #средний-приоритет
> Пользователь отмечает вклад как «safety buffer». Системная цель считает прогресс из баланса вклада, не из транзакций.
---
## Суть
- Пользователь привязывает deposit-счёт к системной цели `safety_buffer`
- Прогресс [[Подушка безопасности]] = баланс вклада / цель
- Цель исчезает из списка когда баланс покрывает target
