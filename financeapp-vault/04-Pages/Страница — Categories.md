# Страница — Categories (Категории)
#страница
> Управление категориями доходов и расходов.

---

## Назначение

Страница отвечает на вопрос: **«По каким направлениям я группирую свои деньги?»**

Категории — основа аналитики. Без них транзакции — просто список чисел. С ними — структура расходов, бюджет, отслеживание приоритетов.

---

## Основные элементы

### Список категорий (CategoriesList)

Карточки категорий с названием, типом и приоритетом. Фильтруются по:
- **Поиск** — по названию
- **Тип** — доход / расход
- **Приоритет** — 5 уровней (см. ниже)

### Приоритеты категорий

Каждая категория расходов получает приоритет — это ключевая классификация для аналитики и бюджетирования:
- **Обязательные (expense_essential)** — расходы, которые нельзя убрать: аренда, коммуналка, еда, транспорт
- **Второстепенные (expense_secondary)** — расходы, которые можно оптимизировать: кафе, развлечения, подписки
- **Целевые (expense_target)** — расходы, привязанные к цели: накопления на отпуск, ремонт
- **Активный доход (income_active)** — зарплата, фриланс
- **Пассивный доход (income_passive)** — проценты, дивиденды, аренда

### Диалог создания/редактирования (CategoryDialog)

Форма с полями: название, тип (доход/расход), приоритет. При создании через форму транзакции — тип подбирается автоматически.

---

## Кнопки и действия

- **Создать категорию** — открывает диалог
- **Редактировать** — клик по карточке
- **Удалить** — с отложенным подтверждением (транзакции с этой категорией станут «Без категории»)
