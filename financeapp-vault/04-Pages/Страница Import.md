# Страница Import
#страница
> /import — загрузка выписок. Двухколоночный layout.
---
## Структура
- Блок «Когда последний раз обновлялись счета» — вверху, свёрнут
- Двухколоночный layout: форма загрузки слева, очередь выписок справа
- Две кнопки: «Загрузить и проверить» и «Добавить в очередь»
## Очередь импорта
- `GET /api/v1/imports/sessions` — список незавершённых сессий
- Клик по сессии → визард для продолжения
## Автораспознавание счёта
PDF-парсер извлекает `contract_number` → предзаполняет + toast «Определён счёт: X»
## Telegram бот
- `@financeapp_import_bot` принимает PDF/Excel/CSV
- Привязка через одноразовый код (/settings → код → отправить боту)
