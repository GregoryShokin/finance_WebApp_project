# Страница — Bank Connections (Подключение банков)
#страница #заглушка
> Автоматическая синхронизация транзакций через банковские API.

---

## Статус: 🔲 Заглушка

Страница существует в маршрутах, но содержит только плейсхолдер. Реализация зависит от доступности банковских API в России.

---

## Планируемое назначение

Страница будет отвечать на вопрос: **«Можно ли не загружать выписки вручную?»**

### Запланированные блоки:
- **Список подключённых банков** — с индикатором статуса (активен / требует переподключения)
- **Кнопка подключения** — инициирует OAuth-авторизацию в банке
- **История синхронизаций** — когда последний раз подтянулись данные, сколько транзакций
- **Переподключение** — если токен банка истёк

---

## Контекст

Сейчас данные попадают в приложение двумя путями: загрузка выписки через [[Страница — Import|Импорт]] или отправка через [[Страница — Settings#Подключение Telegram-бота|Telegram-бот]]. Автоматическая синхронизация через API банков — следующий шаг, который значительно упростит использование.
