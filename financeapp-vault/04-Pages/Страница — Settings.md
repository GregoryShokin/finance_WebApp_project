## Основные элементы

### Подключение Telegram-бота

Привязка аккаунта к Telegram-боту `@financeapp_import_bot` для отправки выписок через мессенджер.

**Процесс подключения:**
1. Пользователь нажимает «Получить код»
2. Система генерирует одноразовый код с ограниченным временем действия
3. Пользователь копирует код (кнопка «Скопировать») и отправляет боту
4. Статус подключения обновляется автоматически

**Состояния:**
- Не подключён — кнопка «Подключить»
- Подключён — иконка связи + кнопка «Отключить»
