## Разобранные гипотезы (2026-04-19)

### ❌ ДДС-модель потоков — отклонено

Расширение модели потоков до CFO/CFF/CFI (корпоративный стандарт Cash Flow Statement). После стресс-теста выбрано простое решение — трёхслойная модель с балансовой замкнутостью через строку-компенсатор, без явных CFO/CFF/CFI на уровне UX.

**Документы:**
- [[Гипотеза — ДДС-модель потоков]] — статус: отклонено, см. § 0 «Итог разбора»
- [[Гипотеза — Стресс-тест ДДС-модели]] — архив; часть 3 (метрика «Доступно для целей») неактуальна
- [[Гипотеза — UX виджета ДДС]] — ✅ актуализирована для трёхслойной модели (название историческое)

**Ключевые попутные решения, которые пошли в действующую спецификацию:**
- Проценты по кредитам = regular `expense` в системной категории (не CFF)
- Вклады = ликвидные счета, не «инвестиции»
- Окно средних для всех метрик = 12 мес.
- Строка-компенсатор «Прирост долга по кредиткам» в декомпозиции Полного
- Прогрессивное открытие табов Базовый → Свободный → Полный по модулям школы
- Подсказка-наставник в красной зоне с планом выхода

### ❌ Свободный остаток — отклонено

Метрика «сколько рублей можно безопасно отправить на цель прямо сейчас». Принципиально ломается на кредитных операциях (disbursement инвертирует формулу). Решение — **не делать единое число**, объяснять через связку показателей (Свободный поток + DTI + Буфер устойчивости + Капитал) в рамках школы.

**Документы:**
- [[Гипотеза — Свободный остаток]] — статус: отклонено, см. § 0
