## Формат уроков

### Структура: Боль → Понимание → Действие → Результат

1. **Боль (Зацепка)** — ситуация, которую пользователь узнаёт. Из жизни, не из учебника
2. **Понимание (Концепция)** — один принцип, с аналогией. Один термин за раз
3. **Действие (Практика)** — конкретное задание в приложении, прямо сейчас
4. **Результат (Награда)** — что изменится. Привязка к виджету

> Для практических/настроечных уроков (0.2, 0.3, 0.4): вместо «боли» — вступление с объяснением зачем нужен этот шаг.

### Зацепки (hooks)

Используем кейсы с конкретными числами, НЕ риторические вопросы.

### Текстовые блоки

Под каждой частью видео — текстовый блок с выжимкой. Две задачи: для тех кто предпочитает читать, и как справочник при возврате к уроку.

### Формат видео

~80% видео (split-screen: спикер + демо в приложении), остальное — квизы/интерактив.

## Механика квизов

### Мини-квиз после каждого урока
- 1–2 вопроса
- После прохождения → разблокируется виджет/раздел
- Не прошёл → пересдача, виджет не разблокируется

### Финальный квиз модуля
- 8–10 вопросов по всем урокам модуля
- Проходной балл: 8 из 10
- Вопросы мини-квизов и финального НЕ пересекаются
- Пересдача: неограниченные попытки, вопросы и варианты перемешиваются

### Оценка финального квиза

| Балл | Результат | Следующий модуль |
|------|-----------|-----------------|
| 10/10 | «Отлично! Полностью усвоил» | ✅ Открыт |
| 8–9/10 | «Хороший результат» | ✅ Открыт |
| 6–7/10 | «Есть пробелы, пересмотри уроки [список]» | ❌ Закрыт |
| ≤5/10 | «Нужно повторить материал» | ❌ Закрыт |

### Место квиза модуля 0

Финальный квиз модуля 0 появляется **после оплаты курса**, не до.

## Принцип построения модулей

### Обязательные уроки
Универсальные финансовые знания для каждого пользователя. Понимание показателей, терминов, механик.

### Необязательные уроки
Для тех, у кого есть конкретная ситуация прямо сейчас. Пользователь без кредитов может пропустить урок «Высокая кредитная нагрузка: план выхода».

> Это решает проблему «пирамида загоняет в рамки». Все проходят один курс, но каждый берёт то, что ему нужно.

## Работа с откатами

Ни один финансовый курс не работает с провалами системно. Мы — работаем.

- Нормализация заранее: «Ты сорвёшься. Это "когда", не "если"»
- Не стыдим за паузу
- Протокол возврата: загрузи выписку → посмотри [[СМО]] → найди одну причину → одна маленькая цель
- Push через 14 дней неактивности

## Тон и язык

**Делай:** «Ты», не «вы». Числа в рублях. Аналогии из быта. Один совет, не пять.
**Не делай:** Не начинай с определения. Не используй термины без объяснения. Не морализируй. Не обещай лёгкости.

### Введение термина
Формула: «[Бытовое объяснение] — это называется [термин]. [Пример с числами].»

---

## Чему мы НЕ учим

### Жёсткое «нет»
Трейдинг, криптовалюта, сетевой маркетинг, микрозаймы.

### Мягкое «не сейчас»
Отдельные акции, недвижимость как инвестиция, сложные инструменты.

> Школа несёт ответственность за то, чему учит. Лучше не научить, чем научить плохо.
