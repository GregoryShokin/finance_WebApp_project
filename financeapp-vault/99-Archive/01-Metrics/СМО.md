# СМО

#показатель #архив

> **Статус: АРХИВ (апрель 2026).** Заменён мультиметрической системой. См. [[Поток]], [[Капитал]], [[DTI]], [[Запас]], [[FI-score]].

---

## Почему архивирован

СМО (среднемесячный остаток) служил единственным фундаментом «Пирамиды финансовых приоритетов». По мере проработки кредитной логики выяснилось, что попытка уместить все финансовые операции в одну метрику порождает каскад исключений: пять классификаций поступлений, два режима кредитки, FIFO-очереди для рассрочек, исключение крупных покупок, тегирование потребкредитов.

Решение: вместо одной метрики — четыре, каждая отвечает на свой бытовой вопрос. Подробный анализ и стресс-тест: [[Мультиметрическая система — стресс-тест]].

## Что пришло на замену

| Было (СМО) | Стало |
|---|---|
| «Сколько остаётся?» | [[Поток]] — базовый (регулярные) + полный (всё по счетам) |
| «Я богатею или беднею?» | [[Капитал]] — активы + кэш − долги |
| «Сколько я работаю на банк?» | [[DTI]] — платежи по кредитам / регулярный доход |
| «Сколько продержусь без дохода?» | [[Запас]] — доступный кэш / средний отток (в месяцах) |

## Что сохранилось от СМО

Ключевые принципы, заложенные в СМО, перешли в мультиметрику:
- Разделение регулярного и нерегулярного — теперь через [[Флаг регулярности на категориях]]
- Поток vs баланс — теперь формализовано: Поток = реальные движения кэша по счетам
- «Планируй по стабильному доходу» — базовый поток считается только из регулярных категорий

## Исходная философия (для справки)

СМО — метрика потока, не баланса. Отвечала на вопрос: «Сколько своих денег у меня остаётся каждый месяц?»

Формула: `СМО = Σ (регулярные доходы − расходы) за N завершённых мес. / N`

Подробная история решений: [[Классификация типов поступлений]], [[Крупные покупки — ход обсуждения]], [[Отвергнутые подходы]].
