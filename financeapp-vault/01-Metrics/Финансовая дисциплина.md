# Финансовая дисциплина

#показатель

> **Актуально на:** 2026-04-16 | Источник: -> [[Спецификация — Целевое состояние системы]]

> Насколько пользователь следует своему бюджету. Рассчитывается по 4 направлениям, только для завершённых месяцев.

> [!note] Роль в системе
> Финансовая дисциплина — **не** одна из первичных метрик ([[Поток]], [[DTI]], [[Капитал]], Буфер устойчивости, [[FI-score]], [[Норма сбережений]]). Это вспомогательный показатель, используемый на странице Health и в Planning. В [[FI-score]] входит косвенно (через страницу Health), но не является прямым компонентом формулы с целевыми весами. Запас удалён из мультиметрики (v1.2).

---

## Формула

Взвешенное среднее по объёму плана по 4 направлениям:
1. Доходы активные
2. Доходы пассивные
3. Обязательные расходы
4. Второстепенные расходы

### Правила

- Для доходов: выполнение = `min(факт/план, 1)`
- Для расходов: выполнение = `min(план/факт, 1)`
- Направления без бюджета **не учитываются**
- Только завершённые месяцы

## Где используется

- [[Страница Health]] — тепловая карта + хронические нарушения + траты без плана
- [[FI-score]] — косвенно (через Health), не прямой компонент формулы
- [[Страница Planning]] — основной показатель выполнения бюджета
