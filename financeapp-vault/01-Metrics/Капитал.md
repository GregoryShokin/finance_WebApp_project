## Формула

```
Капитал = Сумма ликвидные активы (regular, cash) + Сумма вклады (deposit) - Сумма долги по кредитам
```

- **Ликвидные активы:** деньги на счетах типа `regular` и `cash`
- **Вклады:** счета типа `deposit`
- **Долги:** остаток по кредитам + задолженность по кредиткам + остаток по рассрочкам

### Без неликвидных активов

Квартира и машина — не в формуле Капитала. Причина: для ЦА с DTI ~42% неликвидный актив — шум. Квартиру нельзя продать, чтобы заплатить за продукты.

Расширенный показатель [[Чистый капитал]] (с неликвидами) — отдельно на странице Assets.

## Главная ценность: траектория, не позиция

Пользователь видит не «-150к» (демотивирует), а **«растёт на 8к/мес ->»** (мотивирует). Направление важнее абсолютного числа.

Тренд капитала за 3-12 мес. — ключевой компонент [[FI-score]] с весом 0.30 (наибольший среди четырёх компонентов).

## Что ловит

- **Долговая спираль** — капитал падает при хорошем потоке
- **Lifestyle inflation** — капитал стоит на месте при хорошем доходе
- **Истощение накоплений** — капитал снижается, запас тает

## Роль разделения тело/проценты

Для Капитала разделение тело/проценты в кредитных платежах **важно**:
- Тело (`transfer`) уменьшает долг -> улучшает Капитал
- Проценты (`expense`) — чистый расход -> не влияют на Капитал напрямую

## Где используется

- [[Страница Dashboard]] — виджет «Капитал» (число + тренд + стрелка). Переключатель: ликвидный / чистый
- [[Страница Assets]] — сводка по табам
- [[FI-score]] — компонент «Капитал (траектория)» с весом 0.30 (v1.4)
