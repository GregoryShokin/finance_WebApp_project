# Чистый капитал

#показатель

> **Актуально на:** 2026-04-16 | Источник: -> [[Спецификация — Целевое состояние системы]]

> Расширенный вариант [[Капитал|Капитала]], включающий неликвидные активы. Только на странице Assets, **не на Dashboard**.

---

## Формула

```
Чистый капитал = Капитал + неликвидные активы
```

Развёрнуто:
```
Чистый капитал = Сумма regular.balance + Сумма cash.balance + Сумма deposit.balance + Сумма RealAsset.estimated_value - Сумма credit.balance
```

- Вклады (`deposit`) включаются как актив, отдельной строкой в виджете
- Неликвидные активы — оценочная стоимость имущества (квартира, машина и т.д.)
- Кредитные счета — только остаток долга

## Отличие от Капитала

| | [[Капитал]] | Чистый капитал |
|---|---|---|
| Неликвидные активы | Нет | Да |
| Где отображается | Dashboard, Health, FI-score | Только Assets |
| Для кого важнее | ЦА «Первый шаг» (DTI 42%) | ЦА «Прорыв» (есть имущество) |

Причина разделения: для ЦА с DTI ~42% неликвидный актив — шум. Квартиру нельзя продать, чтобы заплатить за продукты. Поэтому на дашборде — только ликвидный [[Капитал]].

## Где используется

- [[Страница Assets]] — сводка по табам
- [[Страница Dashboard]] — переключатель «ликвидный / чистый» в виджете Капитала (дополнительный режим)
