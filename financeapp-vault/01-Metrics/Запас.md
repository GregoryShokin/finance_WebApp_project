## Формула

```
Запас = доступный ликвидный кэш / средний ежемесячный отток
```

### Числитель: доступный ликвидный кэш

Деньги на дебетовых счетах (`regular`) + наличные (`cash`). **Без** вкладов, **без** неликвидных активов, **без** вычитания долгов.

> [!note] Почему долги не вычитаются
> Долги вычитаются в [[Капитал|Капитале]]. Запас отвечает на другой вопрос: «сколько я протяну на имеющемся кэше». Если вычесть долги из числителя, а платежи по ним включить в знаменатель — получится двойной учёт.

### Знаменатель: средний ежемесячный отток

Все расходы (регулярные + нерегулярные) + полные кредитные платежи (тело + проценты). Усреднение за 3-6 завершённых месяцев.

> [!note] Почему полные платежи, а не только проценты
> В кризисной ситуации (потеря дохода) человек должен платить банку полный платёж. Тело кредита — это обязательный отток кэша, который реально уменьшает запас.

## Зоны

| Зона | Значение | Рекомендация |
|------|----------|-------------|
| Критично | < 1 мес. | Одна задержка зарплаты — и в долг |
| Минимум | 1-3 мес. | Нужно наращивать |
| Норма | 3-6 мес. | Устойчивая позиция |
| Отлично | > 6 мес. | Можно думать о целях и инвестициях |

## Двухслойное отображение (после реализации целей)

Деньги в целевых накоплениях (goals) по умолчанию **не включаются** в Запас — они «как бы потрачены» при откладывании (см. [[Целевые накопления — модель «Откладывание = расход»]]).

Когда будут реализованы целевые накопления, Запас покажет два числа:

- **Операционный Запас** (без целевых денег) — основной, крупно
- **Полный Запас** (с целевыми) — дополнительный, мелко

```
ЗАПАС
2 мес.              <- операционный (основной)
с целями: 5 мес.    <- полный (подсказка)
```

Школьный момент, когда `операционный < 2 мес. AND полный >= 3 мес.`:
> «Это твой запас в запасе»

## Не путать с Подушкой безопасности

[[Подушка безопасности]] — это **целевая сумма** (3-5 мес. расходов), системная цель. Запас — это **текущая метрика**, сколько месяцев доступного кэша есть прямо сейчас. Подушка = цель. Запас = факт.

## Где используется

- [[Страница Dashboard]] — виджет «Запас» (число месяцев + зона)
- [[FI-score]] — компонент с весом 0.25 (нормализация: 0 мес. -> 0, 6+ мес. -> 10)
- [[Порог финансовой устойчивости]] — условие Запас >= 3 мес.
