## Перечень типов

| Тип | Назначение | В [[Капитал|Капитале]] | В [[Запас|Запасе]] |
|-----|-----------|:----------:|:--------:|
| `regular` | Дебетовые карты, расчётные | + актив | + числитель |
| `cash` | Наличные | + актив | + числитель |
| `deposit` | Вклады | + актив | -- |
| `credit_card` | Кредитные карты | - долг | -- |
| `installment_card` | Карты рассрочки (Халва, Свобода, Сплит) | - долг | -- |
| `credit` | Потребительские, автокредиты, ипотека | - долг | -- |
| `broker` | Брокерские/инвестиционные | бэклог | -- |

## Параметры создания

### regular / cash

| Поле | Обязательное | Описание |
|------|:---:|-----------|
| name | + | Название счёта |
| balance | + | Текущий остаток |
| currency | + | Валюта (дефолт: RUB) |

### credit_card

| Поле | Обязательное | Описание |
|------|:---:|-----------|
| name | + | Название |
| credit_limit | + | Кредитный лимит |
| current_debt | + | Текущий долг |
| interest_rate | + | Годовая ставка (%) |
| grace_period_days | -- | Льготный период |

> [!important] Нет выбора режима учёта
> Единая модель: покупка = расход, погашение тела = перевод. Режим «платёж = расход» отвергнут. См. [[Кредитная логика]].

### installment_card

| Поле | Обязательное | Описание |
|------|:---:|-----------|
| name | + | Название |
| credit_limit | + | Лимит рассрочки |
| current_debt | + | Текущий долг |
| monthly_payment | + | Ежемесячный платёж |

Покупка на карту рассрочки = расход в реальной категории. Ежемесячный платёж = перевод (тело). См. [[Кредитная логика#Карта рассрочки]].

### credit (потребительский / авто / ипотека)

| Поле | Обязательное | Описание |
|------|:---:|-----------|
| name | + | Название |
| original_amount | + | Сумма кредита |
| current_debt | + | Текущий остаток |
| interest_rate | + | Годовая ставка (%) |
| term_months | + | Срок (месяцев) |
| monthly_payment | + | Ежемесячный платёж |
| credit_subtype | -- | `consumer` / `auto` / `mortgage` |

### deposit

| Поле | Обязательное | Описание |
|------|:---:|-----------|
| name | + | Название |
| balance | + | Сумма вклада |
| interest_rate | + | Годовая ставка (%) |
| start_date | -- | Дата открытия |
| end_date | -- | Дата закрытия |
| capitalization | -- | Тип капитализации (daily/monthly/quarterly/yearly) |

### broker

Брокерские (инвестиционные). `investment_buy` / `investment_sell` уже есть в enum транзакций.
-> Бэклог: [[Фича — Инвестиционный счёт]]

## Вспомогательная таблица: InstallmentPurchase

Опциональная детализация покупок по картам рассрочки. **Агрегаты на Account (monthly_payment, current_debt) — первичны.** InstallmentPurchase — необязательное обогащение.

| Поле | Тип | Описание |
|------|-----|-----------|
| id | PK | -- |
| account_id | FK -> Account | Привязка к карте рассрочки |
| transaction_id | FK -> Transaction | Привязка к транзакции покупки |
| description | string | Название покупки |
| category_id | FK -> Category | Категория |
| original_amount | decimal | Полная сумма |
| remaining_amount | decimal | Остаток |
| interest_rate | decimal | Ставка (0% для рассрочки) |
| term_months | int | Срок |
| monthly_payment | decimal | Ежемесячный платёж |
| start_date | date | Начало |
| status | enum | active / closed |
