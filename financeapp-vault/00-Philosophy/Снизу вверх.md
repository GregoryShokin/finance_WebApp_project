# Снизу вверх

#философия #днк-школы

> Принцип №1 ДНК школы. Каждая идея проверяется вопросом: «Поможет ли это семье с DTI 42% и базовым потоком +5 000?»

---

## Суть

Школа — не про хайп. Не про «5 акций, которые сделают тебя богатым». Школа — про то, чтобы помочь реальному человеку с ипотекой + автокредитом + кредиткой взять финансы под контроль.

## Правила

- Если контент работает только для тех, кто уже в порядке — он не для нашей ЦА
- Порядок приоритетов священен: стабилизация → рост → инвестиции. Не перепрыгивай
- Конкуренты начинают с «как заработать больше». Мы начинаем с «куда уходят деньги»

## Проверочный вопрос

> Семья [[Профиль — Первый шаг]] — доход 120 000 на двоих, DTI 42%, базовый поток ≈ +5 000, запаса нет. Поможет ли им этот контент/фича/виджет?
