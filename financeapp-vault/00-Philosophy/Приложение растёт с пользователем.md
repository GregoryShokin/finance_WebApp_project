# Приложение растёт с пользователем

#философия #механика

> Ключевая механика: виджеты открываются по мере прохождения уроков. Сначала знание, потом инструмент.

---

## Как это работает

1. Пользователь регистрируется → видит заблокированные разделы
2. Проходит [[Модуль 0 — Знакомство]] (бесплатный) → получает 6 базовых виджетов
3. Платит за курс → проходит модули 1–5 → виджеты и разделы открываются последовательно
4. Каждый мини-квиз после урока → разблокирует привязанный виджет/раздел

## Модель доступа

### До регистрации
Ничего.

### После регистрации (до модуля 0)
Полное боковое меню, но большинство разделов заблокировано.
Доступны: счета/активы, категории, импорт.
[[FI-score]] виден как число на боковой панели — значение видно, детали нет.

### После модуля 0 (бесплатно)
6 базовых виджетов → см. [[Карта виджетов]]

### Воронка оплаты
Модуль 0 → [[Урок 0.5 — Твой финансовый портрет]] показывает ценность → оплата → финальный квиз модуля 0 → модуль 1.

## Техническая реализация
→ [[Фича — Система разблокировки виджетов]]
- Поле `unlocked_widgets` (JSON array) в модели User
- API: `POST /api/v1/users/unlock-widget`
- 4 виджета доступны сразу без уроков
