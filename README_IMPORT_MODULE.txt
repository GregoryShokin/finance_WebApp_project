Модуль: Импорт транзакций (V1)

Что добавлено:
- backend API для CSV-импорта
- сессии импорта и строки предпросмотра
- preview перед записью в transactions
- commit только после подтверждения
- frontend-страница Импорт с пошаговым сценарием

Как установить:
1. Распакуй архив.
2. Скопируй содержимое архива в корень проекта financeapp с заменой файлов.
3. Примени миграции БД:
   alembic upgrade head
4. Перезапусти backend.
5. Перезапусти frontend:
   cd frontend
   rmdir /s /q .next
   npm run dev

Что появится в системе:
- POST /api/v1/imports/upload
- GET /api/v1/imports/{session_id}
- POST /api/v1/imports/{session_id}/preview
- POST /api/v1/imports/{session_id}/commit

Что поддерживается в этой версии:
- только CSV
- ручное сопоставление колонок
- preview до импорта
- базовая проверка дублей
- импорт только готовых строк

Что пока не входит:
- XLSX
- PDF
- OCR
- история импортов
- редактирование отдельных строк прямо в preview
- автоопределение перевода между своими счетами

Как пользоваться:
1. Создай хотя бы один счёт.
2. Перейди на страницу Импорт.
3. Загрузите CSV-файл.
4. Проверь колонки и сопоставь:
   - дата
   - описание
   - сумма или доход/расход
   - при необходимости категория
5. Построй предпросмотр.
6. Импортируй готовые строки.

Техническое замечание:
- В commit используются уже существующие правила создания транзакций через TransactionService, поэтому балансы счетов обновляются штатно.
